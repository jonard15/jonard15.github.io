Admin Template

The system features role-based access control, allowing admins to manage users, roles, and permissions. Users can be assigned specific permissions and custom menus, ensuring they only access what’s relevant to their role. Login access and system visibility are determined by each user’s assigned role and permissions.

Used Technologies:

  • VueJs
  • Vuetify
  • Laravel 8
  • PostgreSQL
  • MYSQL
